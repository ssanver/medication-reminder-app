# Medicine Catalog Data Source

Bu projede ilaç katalog verisi aşağıdaki kaynaktan alınmıştır:

- Kaynak: https://www.pharmananda.com/medicine-list/
- Yerel CSV dosyası: `docs/pharmananda-medicine-database.csv`
- DB seed mekanizması: `api/services/medicine-catalog-persistence/medicine-catalog-seeder.cs`

Not: CSV içindeki `source_url` kolonu sayfa bazlı orijinal bağlantıları saklar.
